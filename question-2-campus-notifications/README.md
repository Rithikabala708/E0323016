# Campus Notifications Microservice

A real-time notification platform for campus students to receive updates about Placements, Events, and Results.

## Project Structure

```
question-2-campus-notifications/
├── notification_system_design.md    # Complete system design (Stages 1-6)
├── src/                              # Source code (Stage 6 implementation)
├── README.md                         # This file
└── screenshots/                      # Output screenshots (Stage 6)
```

## Stages

- **Stage 1**: REST API Design ✅
- **Stage 2**: Database Schema Design (In Progress)
- **Stage 3**: Query Optimization
- **Stage 4**: Performance & Caching Strategy
- **Stage 5**: Reliable Broadcast System
- **Stage 6**: Priority Inbox Implementation

## Technologies

- **Backend**: Node.js with Express
- **Real-time**: WebSocket (Socket.IO)
- **Database**: PostgreSQL (to be implemented)
- **Caching**: Redis (planned)
- **Authentication**: JWT tokens
